# Contributing to Glynn Cleaner

Thank you for your interest in contributing to Glynn Cleaner.  
This project aims to provide a polished, reliable, and user‑friendly CSV cleaning tool for analysts, developers, and small businesses. Contributions of all kinds are welcome — from bug reports and feature suggestions to code improvements and documentation updates.

---

## How to Contribute

### Reporting Bugs
If you find a bug, please open an issue on GitHub and include:

- A clear description of the problem  
- Steps to reproduce  
- Expected behaviour  
- Actual behaviour  
- Sample input data (if possible)

This helps ensure issues can be diagnosed and fixed quickly.

---

### Suggesting Features
Feature requests are encouraged. When opening a feature request:

- Explain the problem the feature solves  
- Describe how you imagine the feature working  
- Include examples or mockups if helpful  
- Note whether the feature should be optional or default  

All suggestions are reviewed with usability and simplicity in mind.

---
